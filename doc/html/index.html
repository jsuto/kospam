clapf is a modular network filter for postfix. It includes a statistical (inverse chi-square) antispam module, the blackhole feature and antivirus support (clamav, AVG Linux, avast!, Kaspersky, and Dr.Web) preventing virus infection as well as a cgi and a command line interface to easily train the token database. Clapf also features an experimental OCR check function to get rid of image spams.
